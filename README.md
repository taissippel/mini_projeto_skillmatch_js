SKILLMATCH JS 
Simulador de Compatibilidade entre Candidatos e Vagas.

## SOBRE O PROJETO ##
Foi desenvolvido um simulador de compatibilidade de candidatos para vagas de desenvolver front-end.

O simulador tem o objetivo de comparar:
* habilidades dos candidatos;
* requisitos das vagas;
* experiência mínima, para vagas com essa exigência;
* percentual de compatibilidade.

Além disso o projeto gera:
* indicação para o candidato onde ele deve forcar mais os estudos;
* contador de análises;
* simulador de carregamento de vagas;
* relatório resumido de melhor indicado a contratação, para quem estiver realizando a análise de contratação.
